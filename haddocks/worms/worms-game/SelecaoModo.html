Description

Este módulo implementa o ecrã de seleção de modos.

Modos disponíveis:

  • 2 Jogadores
  • VS Bot
  • Treino
  • Voltar ao menu

Cada opção é totalmente navegável com teclado, mantendo consistência com o resto da interface.
