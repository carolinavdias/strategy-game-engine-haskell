Description

Este módulo define o tipo EstadoJogo, que representa todos os ecrãs possíveis da aplicação.

Estados principais:

  • Menu
  • Seleção de Modo
  • Seleção de Mapa (Treino)
  • Jogo em curso
  • Vitória / Game Over
  • Tutorial

Cada estado encapsula os dados necessários para:

  • Desenho independente
  • Processamento de eventos específico
  • Transições de navegação seguras

A utilização de tipos algébricos garante que apenas transições válidas podem ocorrer.
